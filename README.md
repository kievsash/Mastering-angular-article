`git clone <url>`

`cd <project_folder>`

-  1_HTTPInterceptor_in_modules
-  2_Query_params_on_redirect
-  3_ng-deep_to_avoid_styles_incapsulation
-  4_http_call_on_ngInit
-  5_same_route_reload
